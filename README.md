# open-source-framework
集合了开源框架的简单用法


## 0.1为什么要使用开源框架

1. 提高开发速度
2. 提高开发质量

## 0.2选择开源框架的原则

1. 聚合性框架一定要放弃.例如Afinal，xUtils
	* 大而全的框架容易导致牵一发而动全身.可读性差,耦合高,难扩展.

2. last commit超过一年以上或者issues一大堆没有fix的框架不要用.

3. 选择活跃度和人气高的项目
